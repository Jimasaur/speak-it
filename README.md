# Speak It

> "Speak it and it shall be."

A premium, voice-first personal AI assistant service for high-value clients.

## Concept

One agent handles everything — email, calls, scheduling, shopping, finances, community, family logistics. Available via voice, text, email, or web. Always the same context.

## Stack

Built on top of the Leo + Mercury + Grumpy stack:
- **Leo** — OpenAI Realtime voice assistant (Twilio)
- **Mercury** — Rails control plane at mercury.gopherdrones.com
- **Grumpy** — Local OpenClaw reasoning/execution agent
